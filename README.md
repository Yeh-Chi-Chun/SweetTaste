# SweetTaste
 Angular-甜點店
 
# 如何建置
電腦安裝node.js

https://nodejs.org/en/

專案內打開終端機

![image](https://user-images.githubusercontent.com/57256086/154481659-d4c06562-d328-41ab-8971-92c76fcb72f3.png)

1.輸入: npm install -g @angular/cli

2.輸入: npm install

3.輸入: npm start

4.打開網站輸入 http://localhost:4200 即可預覽網站
